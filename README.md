# MatchRox — Community

Public feedback home for the **[MatchRox](https://matchrox.com)** iOS app.

> MatchRox is a partner-matching app for HYROX athletes. This repo isn't the app or its source — it's the place where we gather product feedback at scale.

## What lives here

- **[💬 Discussions](https://github.com/Matchroxx/community/discussions)** — bugs, feature requests, product questions, release announcements, polls
- **[🛡️ SECURITY.md](SECURITY.md)** — how to privately report a vulnerability
- **[📜 CODE_OF_CONDUCT.md](CODE_OF_CONDUCT.md)** — what we expect from everyone participating here

## Where to post what

| Topic | Category | Format |
|---|---|---|
| Release notes from the team | From the Team | Announcement (team-only) |
| New to MatchRox — first-run feedback | Getting Started | Open discussion |
| Feature you wish the app had | Wishlist | Open discussion (upvote to signal) |
| Product polls from the team | Polls | Poll |
| "How do I…?" questions | How do I...? | Q&A (accept an answer) |
| App bug that's annoying but not urgent | Bugs | Q&A (accept a resolution) |

## Where NOT to post

- **Critical app issues** (crashes, billing, can't log in) — use in-app support (Settings → Support)
- **Report another athlete** (harassment, spam, fake profile) — use the in-app report flow on their profile
- **Security vulnerabilities** — see [SECURITY.md](SECURITY.md)
- **Specific account or billing questions** — in-app support, not a public thread

## What happens to your feedback

The team reads every post. Wishlist upvotes directly influence what we build next. When we ship something that came from here, we close the thread and tag it **shipped** so you can see the loop close.
