# Community Code of Conduct

This repository is the public feedback home for the [MatchRox](https://matchrox.com) iOS app. Everyone here — athletes, coaches, staff, contributors — is expected to keep it a useful place for honest product feedback.

## Be decent

- **Focus on the product.** Feedback about the app helps everyone. Personal attacks, pile-ons, and public shaming don't.
- **No harassment.** No slurs, threats, doxxing, or sustained unwanted attention toward anyone — team, athletes, or other posters.
- **Respect athletes' privacy.** Don't post another athlete's name, photo, messages, profile info, or race details without their consent. Redact names in screenshots.
- **Disagree with substance.** "This feature is broken because X" is welcome. "The team is [insult]" is not.
- **Disclose affiliations.** If you're a coach, HYROX employee, gear brand rep, or affiliate, say so when it's relevant to your post.

## Things that belong elsewhere

| If you need to… | Use this instead |
|---|---|
| Report another athlete for harassment, spam, inappropriate content, or a fake profile | **In-app report flow** (profile → Report) |
| Resolve a billing or subscription issue on your account | **In-app support** (Settings → Support) |
| Report a security vulnerability | [SECURITY.md](SECURITY.md) |
| Get medical advice | Your doctor, PT, or sports medicine pro — not strangers on the internet |

## Enforcement

The MatchRox team moderates this space. Depending on severity, we may:

1. Edit or delete the post
2. Comment with a correction and a link to this document
3. Temporarily block the account from posting
4. Permanently block the account
5. Report the GitHub account to GitHub's Trust & Safety team

For serious violations we'll skip straight to step 4 or 5.

## Reporting

If you see something that violates this code and the team hasn't caught it yet, email `trust@matchrox.com` with a link to the post. Don't @-tag or reply-chain call-outs — those usually make things worse.
